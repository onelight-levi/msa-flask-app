# cicd training

## 원본
https://github.com/worldvit/your_flask_app.git

## 실습 방식
Monolithic으로 구현된 웹 서비스를 기능별로 나눠 service와 pod를 나누는 MSA 방식으로 리팩토링하고 git action과 AWS를 이용해 CI/CD를 구현하는 것을 목표로 한다.
